## Functionality
- Displaying posted jobs
- Pagination of posts
- Displaying single post
- Mobile responsiveness


## Setup
 
Install all dependencies:

```
npm install
```


To run application in development mode:

```
npm run dev
```

To create production build:

```
npm run build
```
